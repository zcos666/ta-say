# 《过拟合恋人》项目工作台

《过拟合恋人》是一个“前半段恋爱 Meta 恐怖互动作品 + 后半段恋爱语言翻译器”的网页项目。

当前仓库的唯一产品基准是 `ta说_PRD_最终版.md`。  
所有说明文档、协作提示、模块 README 和后续实现要求，都必须服从该 PRD。

## 当前定位

- 当前开发定位：正式开发版
- 不以 `MVP`、`Demo`、`mock 闭环` 作为验收目标
- 不接受静态页、假接口、预设结果等实质功能代偿

## 核心能力

- 关系恐惧与 `TA` 称呼选择
- 聊天主界面
- 反义污染
- 恐怖回复系统
- 输入框监听
- 读档 / 存档系统
- 语音输入
- 空间异常
- 退出异常
- 真相揭示与梦醒
- 恋爱翻译官
- 分享卡导出

## 开发原则

1. 剧情推进靠规则系统。
2. 文本细节靠 LLM。
3. 关键正式功能不得由假数据、预设文本或静态展示代替。
4. 页面组件不直接承担模型请求。
5. 文档与实现冲突时，以 PRD 为准。
