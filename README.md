# 《过拟合恋人》项目工作台

《过拟合恋人》仍然是一个“前半段恋爱 Meta 恐怖互动作品 + 后半段翻译器能力”的网页项目。

当前阶段不推翻原有大框架，但开发重点已经收敛为两条并行主线：

- 翻译器分析：基于历史聊天记录输出关系分析报告
- 蒸馏能力：蒸馏自己与蒸馏别人，为后续更强的个性化能力做准备

游戏主线继续保留，但不作为当前唯一开发重心。

## 当前定位

- 当前开发定位：正式开发版
- 当前重点：翻译器分析主线 + 蒸馏能力探索线
- 不接受静态页、假接口、预设结果等实质功能代偿

## 当前主能力

### 保留能力

- 关系恐惧与 `TA` 称呼选择
- 聊天主界面
- 反义污染
- 恐怖回复系统
- 输入框监听
- 读档 / 存档系统
- 空间异常
- 真相揭示与梦醒

### 当前重点能力

- 聊天记录分析型翻译器
- 关系分析报告
- 蒸馏自己
- 蒸馏别人

## 开发原则

1. 游戏世界观保留，但当前开发重心优先让翻译器和蒸馏能力落地。
2. 剧情推进靠规则系统，分析和文本生成由模型能力承担。
3. 翻译器主输出应是关系分析，不应退化成普通单句回复器。
4. 关键正式功能不得由假数据、预设文本或静态展示代替。
5. 页面组件不直接承担模型请求。
