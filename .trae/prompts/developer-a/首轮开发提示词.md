# 开发者 A 首轮开发提示词

请基于《ta说》现有仓库结构，只完成聊天主链路基础能力：

目标：

1. 搭建 `ChatPage` 骨架。
2. 实现用户发送消息。
3. 第 3 次发送必触发第一次污染。
4. 污染逻辑先使用本地规则，不接真实 API。

范围：

- `src/pages/ChatPage`
- `src/features/pollution`
- `src/types`

限制：

1. 不要改翻译官相关页面。
2. 不要引入新状态管理库。
3. 不要直接操作 `localStorage`。

验收：

- 能发送消息
- 能显示污染前后文本
- 状态结构清晰，可继续接 TA 回复系统
