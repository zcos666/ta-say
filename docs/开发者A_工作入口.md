# 开发者 A 工作入口

## 1. 你的职责

你负责《过拟合恋人》的主剧情体验：

- 开始页
- 选择页
- 聊天主界面
- 反义污染
- 恐怖回复系统
- 输入框监听
- 读档系统
- 空间异常
- 真相页
- 梦醒页

## 2. 你优先进入的目录

- `src/pages/StartPage`
- `src/pages/SelectionPage`

- `src/pages/WakePage`

## 3. 你的工作要求

1. 按正式功能要求实现聊天主链路。
2. 不允许以预设回复或静态逻辑长期代替正式回复能力。
3. 剧情推进继续由规则系统控制。
